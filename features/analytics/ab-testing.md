# A/B Testing

## Définition
Tester deux versions d'un élément (titre, CTA, layout) pour mesurer laquelle convertit le mieux.

## Sous-features
- [ ] Split du trafic (50/50, 70/30, etc.)
- [ ] Variantes de page/composant
- [ ] Tracking des conversions par variante
- [ ] Signifiance statistique
- [ ] Auto-winner (activer automatiquement la variante gagnante)

## Questionnaire de cadrage
1. Quels éléments tester ? (titre, CTA, image, layout, prix)
2. Quel outil ? (Google Optimize successor, PostHog, VWO, custom)
3. Quel est l'objectif de conversion pour chaque test ?
4. Quel volume de trafic pour atteindre la signifiance statistique ?
5. Faut-il un auto-winner ou une décision manuelle ?
