# Drill-down

## Définition
Navigation dans les données du général au détail. Cliquer sur un KPI ou un graphique pour voir les données sous-jacentes.

## Sous-features
- [ ] Clic sur KPI → graphique détaillé
- [ ] Clic sur graphique → tableau des données
- [ ] Breadcrumb de navigation (retour aux niveaux supérieurs)
- [ ] Filtres contextuels (le drill-down applique un filtre automatique)

## Questionnaire de cadrage
1. Quels éléments sont drilldownables ? (KPIs, graphiques, cellules de tableau)
2. Quel est le chemin de drill-down ? (ex: total → par mois → par projet → par transaction)
3. Faut-il un breadcrumb pour naviguer entre les niveaux ?
4. Le drill-down ouvre-t-il une nouvelle page ou un panneau/modal ?
5. Combien de niveaux de profondeur ?
