# KPIs et métriques

## Définition
Indicateurs clés de performance affichés dans un dashboard pour suivre l'activité et la santé du business.

## Sous-features
- [ ] Cards KPI (nombre + variation + icône)
- [ ] Variation par rapport à la période précédente (%, delta)
- [ ] Indicateur de tendance (flèche up/down)
- [ ] Code couleur (vert/rouge selon performance)
- [ ] Sélecteur de période
- [ ] Objectifs / targets
- [ ] Sparkline (mini-graphique dans la card)

## Questionnaire de cadrage
1. Quels KPIs afficher ? Lister chacun avec sa formule de calcul.
2. Faut-il montrer la variation par rapport à la période précédente ?
3. Quelle période par défaut ? (aujourd'hui, semaine, mois, année)
4. L'utilisateur peut-il changer la période ?
5. Y a-t-il des objectifs/targets à comparer ? (ex: objectif mensuel de collecte)
6. Faut-il un code couleur selon la performance ? (vert si objectif atteint)
7. D'où viennent les données ? (base SQL, API, agrégation temps réel)
8. À quelle fréquence les KPIs sont-ils mis à jour ?
