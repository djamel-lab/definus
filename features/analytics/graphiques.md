# Graphiques interactifs

## Définition
Visualisations de données sous forme de graphiques interactifs dans un dashboard.

## Sous-features
- [ ] Line chart (évolution temporelle)
- [ ] Bar chart (comparaison)
- [ ] Pie / donut chart (répartition)
- [ ] Area chart (volume cumulé)
- [ ] Stacked bar (composition)
- [ ] Gauge / jauge (progression vers objectif)
- [ ] Scatter plot (corrélation)
- [ ] Heatmap (densité)
- [ ] Carte géographique (choropleth)
- [ ] Tooltip au survol
- [ ] Légende interactive
- [ ] Zoom / pan
- [ ] Export image (PNG, SVG)
- [ ] Responsive

## Questionnaire de cadrage
1. Quels graphiques sont nécessaires ? Lister chacun avec les données qu'il affiche.
2. Quel type de graphique pour chaque donnée ? (ligne, barre, camembert, jauge)
3. Faut-il de l'interactivité ? (tooltip, zoom, clic pour détail)
4. Les graphiques doivent-ils être responsives ?
5. Faut-il pouvoir exporter les graphiques en image ?
6. Quelle librairie ? (Recharts, Chart.js, D3, Plotly, Tremor)
7. Faut-il des graphiques en temps réel ? (mise à jour automatique)
8. Y a-t-il une charte graphique à respecter pour les couleurs ?
