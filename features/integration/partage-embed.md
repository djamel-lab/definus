# Partage / Embed

## Définition
Permettre de partager un dashboard ou un graphique via un lien ou un iframe.

## Questionnaire de cadrage
1. Quels éléments sont partageables ? (dashboard complet, graphique individuel, rapport)
2. Le lien est-il public ou protégé par token ?
3. Faut-il un embed (iframe) intégrable dans un autre site ?
4. Le contenu partagé est-il en lecture seule ?
5. Faut-il une expiration sur les liens de partage ?
