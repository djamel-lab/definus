# Permissions extension navigateur

## Définition
Permissions demandées par l'extension au navigateur. Principe du moindre privilège.

## Questionnaire de cadrage
1. Quelles permissions sont strictement nécessaires ? (activeTab, storage, tabs, cookies)
2. Faut-il accéder à toutes les pages ou seulement certains domaines ?
3. Faut-il des permissions optionnelles (demandées à l'usage) vs obligatoires ?
4. Les permissions sont-elles justifiées dans la description du store ?
5. Y a-t-il des données utilisateur accessibles via l'extension ? Comment sont-elles protégées ?
