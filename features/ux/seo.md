# SEO

## Définition
Optimisation pour les moteurs de recherche : meta tags, structured data, sitemap, performance.

## Sous-features
- [ ] Title et meta description par page
- [ ] Open Graph tags (partage social)
- [ ] Twitter Cards
- [ ] Canonical URLs
- [ ] Sitemap XML
- [ ] Robots.txt
- [ ] Schema.org / structured data (FAQ, Organization, Product)
- [ ] H1-H6 hiérarchie correcte
- [ ] Alt text sur les images
- [ ] URLs propres (slugs)
- [ ] Core Web Vitals (LCP, FID, CLS)
- [ ] SSR ou SSG pour le contenu indexable
- [ ] Hreflang (si multilingue)
- [ ] Redirection 301 (anciennes URLs)

## Questionnaire de cadrage
1. Quelles pages doivent être indexées ? Lesquelles exclues ?
2. Faut-il des meta title/description uniques par page ? Qui les rédige ?
3. Faut-il du structured data ? Quels types ? (FAQ, Organization, Product, BreadcrumbList)
4. Le site est-il SSR, SSG ou CSR ? (impacte l'indexabilité)
5. Y a-t-il des URLs à rediriger ? (migration d'un ancien site)
6. Faut-il un sitemap XML automatique ?
7. Y a-t-il des objectifs Core Web Vitals ? (LCP < 2.5s, CLS < 0.1)
8. Le site est-il multilingue ? Faut-il des hreflang ?
