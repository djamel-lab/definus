# CTA (Call to Action)

## Définition
Boutons et éléments qui incitent l'utilisateur à agir : s'inscrire, acheter, contacter, télécharger.

## Questionnaire de cadrage
1. Quel est le CTA principal de chaque page ?
2. Y a-t-il des CTA secondaires ?
3. Le CTA est-il fixe (sticky) ou dans le flux ?
4. Le texte du CTA : quel verbe d'action ?
5. Y a-t-il un A/B test prévu sur les CTA ?
6. Faut-il un CTA flottant sur mobile ? (bouton sticky en bas)
