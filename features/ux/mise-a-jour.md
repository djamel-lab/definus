# In-app updates

## Définition
Mécanisme pour inciter ou forcer les utilisateurs mobile à mettre à jour l'application.

## Questionnaire de cadrage
1. Faut-il forcer la mise à jour pour certaines versions ? (breaking changes API)
2. Faut-il une notification douce (suggestion) ou bloquante (obligatoire) ?
3. Quel est le critère pour forcer ? (version API minimum, faille de sécurité)
4. Faut-il du CodePush / OTA update ? (mise à jour sans passer par le store)
