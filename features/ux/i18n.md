# Internationalisation (i18n)

## Définition
Support multilingue de l'application.

## Sous-features
- [ ] Fichiers de traduction par langue
- [ ] Sélecteur de langue
- [ ] Détection automatique de la langue (navigateur, IP)
- [ ] URL par langue (/fr/, /en/, /ar/)
- [ ] RTL support (arabe)
- [ ] Formats locaux (dates, nombres, devises)
- [ ] Pluralisation

## Questionnaire de cadrage
1. Quelles langues ? (français, anglais, arabe...)
2. Faut-il du RTL (right-to-left) pour l'arabe ?
3. Comment changer de langue ? (sélecteur, URL, auto-détection)
4. Quel système de traduction ? (next-intl, i18next, fichiers JSON)
5. Qui traduit ? (équipe interne, prestataire, IA)
6. Les dates et nombres sont-ils formatés selon la locale ?
7. Les URLs sont-elles localisées ? (/fr/investir vs /en/invest)
