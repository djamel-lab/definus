# Deep linking

## Définition
Liens qui ouvrent directement un écran spécifique dans l'app mobile depuis un email, une notification push, ou un lien web.

## Questionnaire de cadrage
1. Quels écrans doivent être accessibles par deep link ?
2. Faut-il un universal link (iOS) / app link (Android) ? (ouvre l'app si installée, sinon le web)
3. Faut-il gérer le cas où l'app n'est pas installée ? (redirection vers le store)
4. Les deep links sont-ils utilisés dans les emails et les push notifications ?
5. Faut-il du deferred deep linking ? (lien cliqué avant installation, l'app s'ouvre au bon écran après install)
