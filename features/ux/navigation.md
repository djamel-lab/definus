# Navigation

## Définition
Système de navigation de l'application : header, sidebar, breadcrumbs, liens internes.

## Sous-features
- [ ] Header fixe avec logo + menu
- [ ] Sidebar (navigation latérale)
- [ ] Breadcrumbs
- [ ] Menu burger (mobile)
- [ ] Footer avec liens utiles
- [ ] Navigation imbriquée (sous-menus)
- [ ] Navigation par onglets (tabs)
- [ ] Scroll-to-section (ancres)
- [ ] Active state (highlight page courante)
- [ ] Skip to content (accessibilité)

## Questionnaire de cadrage
1. Quel pattern de navigation ? (header seul, header + sidebar, sidebar seule)
2. La navigation est-elle fixe (sticky) ou scrollable ?
3. Combien d'items dans la navigation principale ? Lister.
4. Y a-t-il des sous-menus / navigation imbriquée ?
5. Faut-il un breadcrumb ?
6. Comment la navigation se comporte-t-elle sur mobile ? (burger, bottom tab)
7. Y a-t-il des sections avec navigation par onglets ?
8. Faut-il un footer ? Quels liens ?
